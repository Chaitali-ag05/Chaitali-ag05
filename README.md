## ⭐ Featured Project

### 🎬 Movie Review Sentiment Classifier  
**Python • Scikit-learn • NLTK • Streamlit**

🔗 **Links**  
[GitHub Repository](https://github.com/Chaitali-ag05/Movie-Review-Sentiment-Classifier) | [Live Demo](https://movie-review-sentiment-classifier-njwkfmi6buvpkn6yup7ydg.streamlit.app/)

📌 **Project Overview**

- Built an **NLP model** to classify **50,000+ IMDB movie reviews** as **positive or negative**.
- Performed text preprocessing including **HTML stripping, tokenization, stopword removal, and lemmatization** using **NLTK**.
- Trained and compared multiple classifiers (**Logistic Regression, SVM, Naive Bayes**).
- Achieved **88% accuracy with Logistic Regression**.
- Deployed an **interactive Streamlit web app** for real-time sentiment prediction.
